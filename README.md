# Elevate Digital - Pricing and Services Site

This is the custom-built pricing and services website for Elevate Digital. It is designed to provide clear, transparent pricing packages, components, add-ons, and maintenance plans for clients without relying on external content management systems like WordPress.

## Project Overview

The website serves as a single-page layout (with an additional glossary page) for presenting web development services to potential clients. It aims for a premium, conversion-focused aesthetic with deep attention to typography, spacing, micro-animations, and overall user experience.

- **Main Page (index.html):** Contains the core pricing logic, broken down into specific structural components (Home Page, About Page, Services, Contact, etc.), cost factors, project packages, add-ons, and maintenance tiers.
- **Glossary Page (glossary.html):** A dedicated dictionary designed to demystify technical web development jargon (such as SEO, CMS, API, and DNS) for clients, making communication more transparent.

## Technical Stack

- **HTML5:** Semantic structure and accessibility.
- **CSS3:** Custom properties (variables) for consistent theming, Flexbox/Grid for layout, and modern keyframe animations. Designed for seamless responsiveness across mobile, tablet, and desktop viewports. No external CSS frameworks were used.
- **Vanilla JavaScript:** Powers interactive elements including navigation toggles, smooth scrolling, and dynamic layout adjustments.
- **Phosphor Icons:** Used for scalable, clean vector iconography.

## Design System

The project uses a custom design system centered around a modern, dynamic UI.
- **Colors:** Deep blues (accent) and rich grays against a clean white/light-gray background for maximum contrast and readability.
- **Typography:** Uses the 'Inter' font family for a sleek, modern, and highly legible appearance.
- **Micro-interactions:** Hover states, target highlights, and dynamic dotted underlines are implemented carefully to draw user attention without overwhelming the experience.

## Deployment

To deploy or run the project locally, simply serve the directory using any standard static web server. No complex build tools, package managers, or server-side languages are required.
