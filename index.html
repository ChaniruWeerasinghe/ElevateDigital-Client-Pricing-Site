Admin Roles and Access Control

Who is the admin?

The developer holds Super Admin access. The client gets a Client Admin role.

Role Breakdown

Access Level Developer Client
Source codeFull accessNo access
Server/hostingFull accessNo access
DatabaseFull accessNo access
Domain/DNSFull accessView only (or shared)
Content editingFull accessFull access
Media uploadsFull accessFull access
User managementFull accessLimited (own staff only)
Plugin/theme updatesFull accessNo access
AnalyticsFull accessView only
Payment gateway settingsFull accessView only
BackupsFull accessNo access

Why this matters

  • Developer maintains control over the codebase and infrastructure
  • Client can update their own text, images, and basic content
  • Prevents accidental changes that could break the site
  • If the client wants full ownership, everything can be transferred at project handoff (handover fee applies)
